		1. Class & Object
			"class" is a keyword used to create collection of declarations like variables declarations,
			functions, constants and further classes

			Access Modifiers
			=================
			1. public 		: give access anywhere
			2. private		: give access only inside class
			3. protected		: give access only inside class and to specific other classes

			class ClassName
			{
				<AccessModifier> variable;
				<AccessModifier> variable;

				<AccessModifier> Function(){}
				<AccessModifier> Function(){}
			}

			Note: In Class Default Access Modifier is "public" if user not specify
			


			"Object" means reference also called as Instance of Class or Entity of a class

			Note: For a class, You can create any number of Objects
